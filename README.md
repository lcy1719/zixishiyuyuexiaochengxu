# zixishiyuyuexiaochengxu
本系统包含管理员和用户2个角色。 1.管理员角色： 用户管理：管理系统中所有用户的信息，包括添加、删除和修改用户。 公告管理：管理公告信息，包括添加、修改和删除公告。 自习室信息管理：管理自习室的信息，包括添加、修改和删除自习室信息。 留言板管理：管理留言板的内容，包括查看、修改、删除和回复留言。 轮播图管理：管理轮播图的配置参数。 预约自习室管理：管理预约自习室的信息，包括查看预约情况。 2.用户角色： 个人信息管理：用户可以管理自己的个人信息，包括账号、密码、手机、学号、姓名和头像。 自习室信息查看：用户可以查看自习室的信息。 预约自习室：用户可以预约自习室，并查看预约情况。 留言板管理：用户可以查看和新增留言。
